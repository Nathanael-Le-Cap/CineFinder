# CineFinder

Une application web moderne pour explorer les films populaires et rechercher des titres spécifiques, utilisant l'API TheMovieDB.

![Aperçu du projet](https://via.placeholder.com/800x400?text=Ajouter+une+capture+d'ecran+ici)

## Fonctionnalités

- **Découverte :** Affichage des films populaires du moment.
- **Recherche :** Barre de recherche en temps réel avec "debounce" (anti-spam API).
- **Favoris :** Système de "J'aime" persistant (sauvegardé dans le navigateur).
- **Responsive :** Interface adaptée aux mobiles et ordinateurs.

## Stack Technique

Ce projet a été réalisé pour démontrer la maîtrise de l'écosystème React moderne :

- **Framework :** React 18 (avec Hooks)
- **Langage :** TypeScript (Typage strict des données API)
- **Build Tool :** Vite (Performance)
- **Styling :** Tailwind CSS (Design responsive et utilitaire)
- **HTTP Client :** Axios
- **Icons :** Lucide React
